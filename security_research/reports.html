Exploit:Chromium/canvas.screenlocker

Target: Any Chromium browser

Severity: Low-Medium

Description: Websites can abuse web APIs to lock the user into fullscreen. As of currently, this attack is low-medium severity due to the inability to block the use of the esc key, however, this attack is being used in tech support scams to pretend to be an antivirus or OS security alert.

Tested: Yes

Live exploits: 1

User interaction required: Yes

What users can do: Use antimalware tools, disable JavaScript on untrusted websites. If attacked, the user can press esc and close the tab. The difficulty of detecting this attack depends a lot about how the attack is done.
